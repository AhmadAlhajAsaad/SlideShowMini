# SlideShowMini

SlideShowMini is a small and simple image slideshow application created using **HTML**, **CSS**, and **JavaScript**.
It displays a main image at the top and allows the user to navigate between multiple images using navigation buttons.
This project demonstrates the use of arrays, functions, and DOM manipulation in JavaScript.

## Features
- Display a main image with navigation buttons
- Switch between images manually
- Uses JavaScript arrays and functions to manage image transitions
- Clean and simple user interface

## Purpose
This project was created to practice:
- JavaScript basics
- Working with arrays
- Functions and DOM manipulation
- Simple front-end UI design

## How to Use
1. Open `index.html` in your browser.
2. Use the navigation buttons to move through the images.
